1. Discuss the features of Python.

Python is a high-level, interpreted, and general-purpose programming language that emphasizes code readability and simplicity. One of its most notable features is its clean and easy-to-understand syntax, which allows developers to write fewer lines of code compared to other programming languages like Java or C++. This makes Python especially suitable for beginners.

Key Features of Python:

  1. Easy to Learn and Use: Python’s syntax is designed to be intuitive and similar to English, making it beginner-friendly.

  2. Interpreted Language: Python code is executed line-by-line, which makes debugging easier.

  3. Dynamically Typed: You don’t need to declare data types explicitly; Python determines the type automatically at runtime.

  4. Extensive Standard Library: Python provides a wide range of modules and functions for various tasks like regular expressions, file I/O, web services, and more.

  6. Object-Oriented and Procedural: Python supports both object-oriented programming (OOP) and procedural styles.

  7. Embeddable and Extensible: Python can be embedded in C/C++ programs and extended with other languages.

  10. Used in Diverse Fields: Python is used in web development, data science, machine learning, automation, networking, and more.

In summary, Python’s simplicity, flexibility, and vast ecosystem make it a powerful language for both beginners and professionals. It is widely used in both academic and industrial settings.

6. Explain various decision-making statements in Python.

Decision-making statements in Python are used to execute specific blocks of code based on certain conditions. Python supports the following decision-making constructs:

  1. if Statement:
    Executes a block if the condition is true.

    python
    age = 18
if age >= 18:
    print("Eligible to vote")

  2. if-else Statement:
    Provides two paths: one if the condition is true, another if false.

    python
    num = 5
if num % 2 == 0:
    print("Even")
else:
    print("Odd")

  3. if-elif-else Statement:
    Allows multiple conditions to be checked.

    python
    marks = 75
if marks >= 90:
    print("A grade")
elif marks >= 75:
    print("B grade")
elif marks >= 60:
    print("C grade")
else:
    print("Fail")

  4. Nested if Statement:
    if blocks inside another if block.

    python
    a = 10
b = 5
if a > 0:
    if b > 0:
        print("Both numbers are positive")

These statements allow a program to make decisions and respond accordingly. Python’s use of indentation (instead of braces) makes the structure of these conditionals easy to read.

7. Illustrate different loops in Python with example programs.

Python supports three main types of loops: for, while, and nested loops. These are used to execute a block of code repeatedly.

  1. for Loop:
    Used to iterate over a sequence like a list, tuple, or range.

    python
    for i in range(5):
    print(i)

  2. while Loop:
    Repeats a block as long as a condition is true.

    python
    i = 1
while i <= 5:
    print(i)
    i += 1

  3. Nested Loops:
    Loops inside another loop.

    python
    for i in range(3):
    for j in range(2):
        print(i, j)

  4. Loop Control Statements:

    • break: Exit the loop prematurely.

    • continue: Skip the current iteration.

    • pass: Placeholder statement.

    Example using break:

    python
    for i in range(5):
    if i == 3:
        break
    print(i)

These loop structures are powerful tools for repeating tasks, navigating data structures, and implementing complex logic.

8. Illustrate loops with condition at the bottom with an example.

In Python, there is no native do-while loop (which executes once before checking the condition), but we can simulate this behavior using a while True loop with a break condition at the end.

This is useful when the loop body must execute at least once, regardless of the condition.

Simulated do-while loop in Python:

python
while True:
    number = int(input("Enter a number (0 to stop): "))
    if number == 0:
        break
    print("You entered:", number)

Explanation:

  • The loop starts with while True, which makes it an infinite loop.

  • Inside, it asks for user input and processes it.

  • If the user enters 0, the break statement stops the loop.

  • This guarantees that the block runs at least once.

Another example:

python
count = 0
while True:
    print("Count:", count)
    count += 1
    if count >= 5:
        break

This logic mimics the condition-at-bottom behavior found in languages with do-while constructs. It is helpful when the termination condition can only be checked after an iteration.

In Python, the open() function is used to open a file. It accepts a mode argument that determines the operation to perform on the file (read, write, append, etc.).

Common File Modes:

  1. 'r' – Read (default mode). Opens the file for reading. If the file doesn't exist, an error is raised.

python
f = open("file.txt", "r")

  1. 'w' – Write. Opens the file for writing. If the file exists, its content is overwritten; if it doesn't, a new file is created.

python
f = open("file.txt", "w")

  1. 'a' – Append. Opens the file for appending. Data will be written at the end of the file, preserving existing content.

python
f = open("file.txt", "a")

  1. 'b' – Binary. Used for reading or writing binary files.

python
f = open("image.jpg", "rb")

  1. 'x' – Exclusive creation. Creates a new file, but if the file already exists, it raises a FileExistsError.

python
f = open("newfile.txt", "x")

  1. 't' – Text (default mode). Used for text files (default).

python
f = open("file.txt", "rt")

  1. 'r+' – Read and write. Allows both reading and writing.

python
f = open("file.txt", "r+")

  1. 'w+' – Write and read. Allows reading and writing, overwriting the file if it exists.

python
f = open("file.txt", "w+")

20. Explain the built-in methods available in Python for file handling.

Python provides various methods for reading, writing, and manipulating files.

1. read() – Reads the entire content of the file.

python
f = open("file.txt", "r")
content = f.read()
print(content)
f.close()

2. readline() – Reads one line from the file at a time.

python
f = open("file.txt", "r")
line = f.readline()
print(line)
f.close()

3. readlines() – Reads all lines in the file and returns them as a list.

python
f = open("file.txt", "r")
lines = f.readlines()
print(lines)
f.close()

4. write() – Writes a string to the file. If the file is opened in 'w' or 'a' mode, the content will be written.

python
f = open("file.txt", "w")
f.write("Hello, world!")
f.close()

5. writelines() – Writes a list of strings to the file.

python
f = open("file.txt", "w")
lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
f.writelines(lines)
f.close()

6. close() – Closes the file after the operations are completed.

python
f = open("file.txt", "r")
# Perform file operations
f.close()

7. with statement – Used for automatic file closing.

python
with open("file.txt", "r") as f:
    content = f.read()
    print(content)
# File automatically closed after the 'with' block

The with statement is highly recommended as it ensures the file is closed automatically when the block is exited.
